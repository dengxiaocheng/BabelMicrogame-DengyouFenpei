# Direction Lock: 灯油分配

## One Sentence
玩家用有限灯油照亮工区，光照不足会事故，过亮会引来巡查。

## Core Loop
1. 执行核心循环：查看工区 -> 放置灯 -> 调亮度 -> 工人移动工作 -> 事故/巡查结算 -> 下一夜
2. 按 20 分钟节奏推进：两区照明 -> 移动工人 -> 灯油不足 -> 巡查高压夜

## Must Keep
- 核心机制必须保持：区域覆盖 + 消耗时间 + 巡查注意
- 核心循环必须保持：查看工区 -> 放置灯 -> 调亮度 -> 工人移动工作 -> 事故/巡查结算 -> 下一夜
- 20 分钟结构只作为节奏，不扩成大项目

## Must Not Add
- 不做塔防；核心是光照资源权衡
- 不新增第二套主循环
- 不把小游戏扩成长期经营或开放世界

## Required State
- oil
- light_coverage
- accident_risk
- patrol_attention
- night

## Success
在限定时长内完成主循环，并稳定进入至少一个可结算结局

## Failure
关键状态崩溃，或在本轮主循环中被系统淘汰
